
Dataset used : The dataset used for HealthInsure can be dowloaded from this link : https://www.kaggle.com/datasets/tejashvi14/medical-insurance-premium-prediction

Steps to be taken to run the HealthInsure Application

Step 1. Extract the Source Files from zipped folder
	
	- Extract the contents of the downloaded ZIP file to a directory on your computer.
	- Navigate to the Project Directory where you have extracted the source files.

Step 2. Install Dependencies:

	- Check the version of python installed. The version used for this application is: Python version : 3.12.2. (If you dont have it you can use this link to download from python.org - https://www.python.org/downloads/release/python-3122/ )
	-Run the following commands to install the required dependencies

                    python -m pip install -r requirements.txt


Step 3. Run the Application:
	- Once the dependencies are installed, run the following command to start the Streamlit application:

		            streamlit run app.py

Step 4. Access the Application:
	- Open your web browser and go to the provided URL (http://localhost:8501) if you were not redirected to it automatically.
	- You should see the HealthInsure application interface.
